#### 闭包？
    当内部函数被保存到外部时，将会生成闭包。闭包会导致原有作用域链不释放，造成内存泄漏
#### 1.以下代码执行结果
```
function fn(){
    var arrFn = new Array();
    for(var i = 0; i < 5; i++){
        arrFn[i] = function(){
            return i;
        }
    }
    return arrFn;
}
var box = fn();
for(var j = 0; j < box.length; j++){
    console.log(box[j]());//5 5 5 5 5
}
```
