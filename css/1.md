#####1.div半透明样式如何兼容各种浏览器？
```javascript
<style>
    div{
        width: 100px;
        height: 100px;
        background-color: red;
        opacity: .5;
        filter: alpha(opacity=50);
    }
</style>
```
#####2.margin-top和padding-top的值设置为百分比时是怎么计算的？
    margin-top的百分比是相对于父元素的width,而padding-top是相对于自身的width
