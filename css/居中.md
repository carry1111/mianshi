#### 居中
水平居中
```
针对inline, 内联块inline-block, 内联表inline-table, inline-flex元素及img,span,button等元素
.text_div{
	text-align:center;
}
```
```
不定宽块状元素居中
(1).text-div{
    margin:0 auto;//且需要设置父级宽度
}
(2)父级元素设置
.parent-div{
    display: flex;
    justify-content: center;
}
```
垂直居中
```
单行内联(inline-)元素垂直居中 
通过设置内联元素的高度(height)和行高(line-height)相等，从而使元素垂直居中。

.text-div{
    height: 120px;
    line-height: 120px;
}
```
```
绝对布局方式
已知高度
.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 50%;
  height: 100px;
  margin-top: -50px; 
}
未知高度
.parent {
    position: relative;
}
.child {
    position: absolute;
    top: 50%;
    transform: translateY(-50%);
}
```
```
flex布局
父级元素设置
.parent{
    display: flex;
    align-items: center;
}
```
